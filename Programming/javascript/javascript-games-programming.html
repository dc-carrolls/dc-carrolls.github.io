This course is based on the W3Schools.com HTML Game Example

You will need some programming tools (development environment). The simplist option is to use repl.it but my preference is to use GitHub/GitHub Desktop and VS Code if you are using your own computer and can install them.

You will be able to try out all the code and make your own changes within the W3Schools tutorial, but you will need your own development environment if you want to keep any code and build your own game.

The animation makes use of a Canvas and the setInterval function. Follow the links to find out more about these features.
